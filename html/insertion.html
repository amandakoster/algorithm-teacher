Insertion Sort

Insertion sort is a sorting algorithm that will sort an array of numbers.

Input five numbers ranging from 1 to 100 in random order:


  • [0]
  • [1]
  • [2]
  • [3]
  • [4]

Right off the bat [0] is considered fully sorted.
